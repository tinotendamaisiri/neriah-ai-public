# Sample Exercise Book Images

This directory contains sample images used in the Neriah Kaggle demo notebook.

## Required images

| File | Description |
|---|---|
| `question_paper.jpg` | A real question paper from a Zimbabwean school (Form 2 Mathematics or Science) |
| `student_submission.jpg` | A real student's handwritten exercise book page with answers |
| `student_submission_2.jpg` | A second student's work (different handwriting, for comparison) |

## Status

These images are placeholders generated by `create_placeholders.py`.
Replace them with real exercise book photos from Kundai before final submission.

When adding real photos:
- Obtain explicit written permission from the school and parents/guardians
- Remove any identifying information (student names, school name in photo) if required
- Photos should be well-lit, full page, 1000px+ on the short side

## Generating placeholders

```bash
python samples/create_placeholders.py
```

This creates synthetic test images that exercise the full notebook pipeline.
The notebook automatically uses these if no real images are present.
